# AutoBuyerMod Practice Version - Help Files Index

This directory contains comprehensive documentation for learning how to create script mods for Space Haven. The files are organized in chronological order, from initial research through setup, coding, and debugging.

## Documentation Files

### Getting Started
- **[01_Getting_Started.md](01_Getting_Started.md)** - Step-by-step tutorial for setting up your development environment (Maven, AspectJ, IDE configuration)
  - **⚠️ IMPORTANT:** Includes critical information about package naming conventions - read this first!

### Project Structure
- **[02_Project_Structure.md](02_Project_Structure.md)** - Reference guide to understanding the mod directory layout, pom.xml, info.xml, and build configuration

### AspectJ Fundamentals
- **[03_AspectJ_Basics.md](03_AspectJ_Basics.md)** - Reference guide to AspectJ annotations (@After, @Before, pointcuts, advice) and how to hook into game events

### Research Resources
- **[04_Decompiled_Research_Files.md](04_Decompiled_Research_Files.md)** - List of useful classes from the decompiled spacehaven.jar, with descriptions of what each class does

### Game Hooks
- **[05_Game_Hooks_Used.md](05_Game_Hooks_Used.md)** - Detailed list of all AspectJ hooks used in AutoBuyerMod, with explanations of when they fire and what they do

### Core Concepts
- **[06_Core_Concepts.md](06_Core_Concepts.md)** - Reference guide to Space Haven's trade system, World/Ship objects, configuration loading, and other fundamental concepts

### Code Walkthrough
- **[07_Code_Walkthrough.md](07_Code_Walkthrough.md)** - Step-by-step explanation of AutoBuyerMod's key components, how they work together, and why design decisions were made

### Best Practices
- **[08_Common_Patterns.md](08_Common_Patterns.md)** - Reference guide to best practices, error handling, logging, state management, and common coding patterns

### Troubleshooting
- **[09_Troubleshooting.md](09_Troubleshooting.md)** - Common issues and solutions, debugging techniques, and how to diagnose problems

## Learning Path

### For Beginners
1. Start with **01_Getting_Started.md** to set up your environment
2. Read **02_Project_Structure.md** to understand the mod structure
3. Study **03_AspectJ_Basics.md** to learn how to hook into game events
4. Review **05_Game_Hooks_Used.md** to see real examples
5. Read **07_Code_Walkthrough.md** to understand how everything fits together

### For Intermediate Developers
1. Review **04_Decompiled_Research_Files.md** to find useful game classes
2. Study **06_Core_Concepts.md** for deeper understanding of game systems
3. Read **08_Common_Patterns.md** for best practices
4. Reference **09_Troubleshooting.md** when debugging

### For Reference
- Use **04_Decompiled_Research_Files.md** to find game classes you need
- Use **05_Game_Hooks_Used.md** to see hook examples
- Use **06_Core_Concepts.md** for API reference
- Use **08_Common_Patterns.md** for coding patterns
- Use **09_Troubleshooting.md** for debugging help
